# Инструкция по импорту Facebook Reels

## Быстрый способ (Рекомендуется)

### 1. Получите все URL ваших Reels

1. Откройте вашу страницу Facebook: https://www.facebook.com/61582836781257
2. Перейдите в раздел "Reels" или "Видео"
3. Для каждого видео:
   - Откройте Reel
   - Скопируйте URL из адресной строки (например: `https://www.facebook.com/reel/1348545846955296`)

### 2. Добавьте URL в файл

1. Откройте файл `reels-urls.txt` в любом текстовом редакторе
2. Вставьте каждый URL на отдельной строке:

```
https://www.facebook.com/reel/1348545846955296
https://www.facebook.com/reel/2222222222222222
https://www.facebook.com/reel/3333333333333333
```

3. Сохраните файл

### 3. Запустите импорт

```bash
node batch-import-reels.js
```

Все видео будут автоматически добавлены в базу данных и появятся на сайте!

---

## Альтернативный способ через админ-панель

1. Откройте `/admin.html` на вашем сайте
2. Введите URL каждого Reel
3. Нажмите "Добавить видео"
4. Повторите для всех видео

---

## Автоматический способ (требует настройки)

### Через Facebook Graph API

1. Перейдите на https://developers.facebook.com/tools/explorer/
2. Выберите вашу страницу
3. Добавьте разрешения: `pages_read_engagement`, `pages_show_list`
4. Сгенерируйте Access Token
5. Добавьте токен в файл `.env`:

```
FACEBOOK_ACCESS_TOKEN=ваш_токен_здесь
```

6. Запустите:

```bash
node facebook-api-import.js
```

Скрипт автоматически найдет все видео за последние 2 месяца и добавит их в базу.

---

## Проверка результата

После импорта откройте ваш сайт - все видео должны отображаться в секции "Наши успехи".

Если видео не отображаются, проверьте базу данных:
```bash
node -e "import('./check-db.js')"
```
